松哥要升级 SpringBoot 视频了，看看新增了哪些内容！

松哥是去年 7 月份开始录制 Spring Boot 视频的，到现在，有超过 2000 人学习了这套视频教程，能够帮助到小伙伴们，真的是一件很有成就感的事情，比在公司 Coding 有成就感多了。

从去年 8 月份 Spring Boot 视频录制完成，到现在差不多有一年半时间了，这一年半时间也发生了很多事情：

  • Spring Boot 版本从 2.1.6 升级到 2.4.0。
  • 松哥陆续发了很多图文的 Spring Boot 教程，并整理了多个相关的 PDF 文档。
  • 随着在实际项目中继续推进 Spring Boot 技术栈，自认为对 Spring Boot 又有了一些新的理解。
  • …

这些事情综合到一起，就让我不得不考虑是否需要对之前的 Spring Boot 视频教程进行升级？今年从 6 月份到 10 月份一直忙的焦头烂额，没空去考虑这些事情，现在到年底了，稍微轻松一点，在一边和大家分享 ElasticSearch 教程之余，我也一直在琢磨是否需要把 Spring Boot 重新搞一下。

Spring Boot 对我来说是再熟悉不过的技术栈：2019 年初在清华社出版了 Spring Boot 的书；2018 年初开发了 star 数接近 20K 的 Spring Boot+Vue 前后端分离项目（https://github.com/lenve/vhr）；2019 年还写了多个相关的 PDF 教程（公号后台回复 ssm 可以获取）；在实际的项目中一直也是 Spring Boot 技术栈…

所以对于我来说，升级一下之前的 Spring Boot 视频，并不需要花太多时间去准备，主要是录制的过程可能比较费时间，一般只有晚上可能才有空录制。而我的业余生活不可能只有录视频，也有很多其他事情要做，所以一直比较纠结没有下定决心。

最近总算稍微闲一点了，思忖再三，决定利用年前这段时间，对之前的 Spring Boot 视频教程进行一次升级，因为之前部分 Spring Boot 视频有一点点音质问题，所以这次升级不仅仅解决音质问题，还会增加很多新的内容进来，目前想到的有：

  • 基于 Spring Boot2.4.0 录制
  • 日志处理
  • ElasticSearch 整合
  • 异步请求
  • 数据库迁移
  • Spring 批处理
  • …

大家在网上看 Spring Boot 教程的时候，很多人会把 SSM 和 Spring Cloud 扯进来一起讲，松哥之前出过单独的 SSM 教程（公众号后台回复 ssm 可以获取），今年也录制了单独的 Spring Cloud 教程（包含在四套付费视频内），由于 Spring Cloud 教程是今年录制的，目前看起来还是比较新的，因此本次的升级计划主要是针对 Spring Boot。

接下来就是今天的重点了！

所有之前购买过四套付费视频的小伙伴，这次都可以免费学习最新版的 Spring Boot 视频教程！

视频分发形式还和之前一样，录好的视频我会加密后上传到百度网盘，然后会在星球上分享给大家，小伙伴们只需要使用你现有的授权码，就可以打开加密视频播放学习了。

接下来我把视频常见问题给大家罗列一下：

Q1：视频放在哪里，怎么播放？

视频放在百度网盘上，下载后使用专用播放器授权播放，专用播放器一机一码，一人一码。

Q2：视频为什么不支持在线播放？

在线播放的话，一般都是和平台合作，但是国内大家所熟知的平台如慕课网、csdn 学院等，平台抽成基本都是 4 成起，松哥之前还合作了一个平台拿 6 成（也就是假如 100 块钱的课，平台拿走 60）。这样的分成比例无形中会推高课程的价格，现在我自己分发，所以才能做到低价。同时为了防止视频泄露，松哥对视频做了加密处理，加密后的视频只能使用专用播放器播放，因此不支持在线播放。

Q3：以后录的视频都可以看吗？

这次松哥一共计划出四套视频课程，并且四套视频已经全部录完，大家在公众号后台回复 视频目录 可以查看已经录完的视频详细目录。也就是 199 一共可以看这四套视频，但是四套视频问题的解答，会一直持续下去。

Q4：视频可以试看吗？

其实我平时在公号发的视频，大家都可以当作试看的，反正的录视频风格是不变的。小伙伴们也可以在公众号后台回复 vhr试看 获取试看视频合集。

Q5：这套视频适合哪些人看？

学习这套视频，后端需要有 SSM 基础，前端需要有 js 基础（可以不懂 Vue，第十六章视频会讲）。要是不懂 SSM，可以先看松哥之前手敲 5 万余字的 SSM 教程，公众号后台回复 ssm 可以获取下载链接。

Q6：视频中的技术版本是什么

  • 已有的 Spring Boot 视频基于 2.1.6（目前正在升级，新版基于 Spring Boot2.4.0）。
  • 微人事基于 SpringBoot2.1.8。
  • Spring Cloud 基于目前最新稳定版 H 版。
  • Redis 基于 Redis5。
